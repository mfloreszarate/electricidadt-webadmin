# Iconos PWA

Para completar la configuración PWA, necesitas generar los iconos en diferentes tamaños.

Los iconos deben estar en `src/assets/icons/` con los siguientes tamaños:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Herramientas para generar iconos

Puedes usar herramientas online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

O usar herramientas de línea de comandos como `sharp` o `imagemagick`.

## Diseño del icono

El icono debe representar la aplicación ElectricidadT. Puede incluir:
- Un símbolo de rayo o electricidad
- Colores del tema: #667eea (púrpura/azul)
- Fondo sólido o con gradiente

Los iconos deben ser cuadrados y funcionar bien en diferentes tamaños.
